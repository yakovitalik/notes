++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
Краткий блокнот по Git
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

git init							создание репозитория

git add <file>						добавление файла в "отслеживаемые"
git add .							добавление в отслеживаемые всех файлов каталога

git commit -m "Сообщение"			коммит(фиксация) с сообщением
export GIT_EDITOR=vim				Git будет открывать наш текстовый редактор при фиксации изменений

git status							отображает, какие будут внесены изменения при коммите

git config user.name "Виталий Яковлев"		устанавливаем имя автора коммитов
git config user.email yakovitalik@mail.ru 	устанавливаем email автора коммитов

Ну или же можносменить переменные окружения GIT_AUTHOR_NAME и GIT_AUTHOR_EMAIL.
