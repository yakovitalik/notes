=============================================
Создание проекта при помощи Create React App
=============================================

Переходим в репозиторий GitHub:
https://github.com/facebook/create-react-app
Там описаны команды для Node.JS

Клманды для создания:
npx create-react-app my-app
cd my-app
npm start

Переходим в нужную папку и вводим команды.
