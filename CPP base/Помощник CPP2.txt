Заголовки
iostream 		Класс для вывода информации на экран и ввода с клавиатуры
fstream			Класс для ввода-вывода файлов
iomanip			Здесь описаны некоторые манипуляторы
strstream		Классы для работы с объектами "в памяти"
===========================
ifstream		Класс для чтения файла (создается объект этого класса, который 
			инициализиуется именем файла)
ofstream		Класс для записи файла
ostream			Класс выходного потока
istream			Класс входного потока
streambuf		Общий класс буфера
filebuf			Класс буфера
fstream			Общий класс ввода-вывода
ios			Класс, определяющий биты режимов
===========================
РАБОТА С ФАЙЛАМИ
===========================
get()			Чтение символа из файла
put()			Запись символа в файл
rdbuf()			Возвращает указатель на объект класса streambuf(или filebuf)
			ассоциированный с потоковым объектом
write()			Для бинарной записи в файл (метод класса ofstream)
read()			Для чтения бинарного файла (метод класса ifstream)

reinterpret_cast()	Вычисляет адрес буфера типа char
open()			Метод класса fstream, открывающий файл
close()			Закрытие файла(потока)

seekg()			Устанавливает указатель чтения файла
tellg()			Проверяет указатель чтения файла
seekp()			Устанавливает указатель записи файла
tellp()			Проверяет указатель записи файла
beb			начало файла
cur			текущая позиция
end			конец файла

Например:
seekp(-10. ios::end)
Установит указатель записи на 10 байтов до конца файла
***************************
БИТЫ РЕЖИМОВ
***************************
in		Открытие для чтения (по умолчанию для ifstream)
out		Открытие для записи (по умолчанию для ofstream)
ate		Чтение, начиная с конца файла (AT END)
app		Запись, начиная с конца файла (AAPend)
trunc		Обрезать файл до нулевой длинны, если он уже существующий(TRUNCate)
nocreate	Не открывать несуществующий файл
noreplace	Не открывать для вывода существующий файл, если не установлены ate или app
binary		Открыть в бинарном (не текстовом) режиме
