++++++++++++++++++++++++++++
Запуск сервера
++++++++++++++++++++++++++++
В консоли:
php -S localhost:80

Или если нет прав администратора:
php -S localhost:4000

Остановка сервера:
<Ctrl> + <C>

затем в браузере ввести адрес localhost:
http://localhost/
(или же http://localhost:4000/ если запускалось так)
++++++++++++++++++++++++++++

++++++++++++++++++++++++++++
Настройки сервера
++++++++++++++++++++++++++++
php.ini		Файл содержит настройки PHP сервера

Для того, чтобы задать часовой пояс необходимо исправить следующее:
Найти диррективу date.timezone и прописать
date.timezone = 'Europe/Moscow'
! И убрать символ ; в начале строки (так как ; в кофигурационных файлах это комментирование)

Мои примечания по особенностям синтаксиса:
**********************************************************
$		ставится перед каждым использованием переменной
echo, print	вывод строки на экран
//, #		Однострочные комментарии
/*  */		Многострочные комментарии
.		Применяется для конкатенации строк при выводе(как + в Java)
\n		Перевод строки
=>		Оператор доступа для массива
->		Оператор доступа основной(как . в Java)
'текст'		Одинарные кавычки для string

++++++++++++++++++++++++++++
Типы переменных
++++++++++++++++++++++++++++
integer			Целое число
double			Вещественное число(с плавающей точкой)
string			Строка теста
array			Ассоциативный массив(с ключами) (ключ=>значение)
object			Ссылка на объект. Доступ к свойствам и методам ->
resource		Ресурс(Например дескриптор открытого файла)
boolean			Логический тип
null			Специальное значение
callable		Функция обратного вызова
++++++++++++++++++++++++++++
Действия с переменными
++++++++++++++++++++++++++++
isset()			Проверяет, существует ли(инициализирована ли) переменная
unset()			Уничтожение переменной(делает неинициализированной). Чаще применяется
			для удаления элемента в ассоциативном массиве.
++++++++++++++++++++++++++++
Определение типов переменных
++++++++++++++++++++++++++++
is_int($a)		Возвращает true, если $a целое число
is_double($a)		Возвращает true, если $a действительное число
is_infinite($a)		Возвращает true, если $a бесконечное действительное число INF
is_nan($a)		Возвращает true, если $a недопустимое чмсловое значение NAN
is_string($a)		Возвращает true, если $a является строкой
is_numeric($a)		Возвращает true, если $a является числом или строковым представление числа
is_bool($a)		Возвращает true, если(и только если) $a имеет значение true или false
is_scalar($a)		Возвращает true, если $a — один и перечисленных выше типов. То есть, если это —
			простой тип (скалярный)
is_null($a)		Возвращает true, если $a хранит значение null.
is_array($a)		Возвращает true, если $a является массивом
is_object($a)		Возвращает true, если $a содержит ссылку на объект
gettype($a)		Возвращает строки, соответственно, со значениями: "array", "object", "integer",
			"double", "string", "boolean", "null" и т. д. или "unknown type" в зависимости от типа
			переменной.
++++++++++++++++++++++++++++
Установка типа переменной
++++++++++++++++++++++++++++
settype($var, $type)	Функция пытается привести тип переменной $var к типу $type ($type — одна из строк,
			возвращаемых gettype(), кроме boolean). Если это сделать не удалось (например, в $var
			"нечисловая" строка, а мы вызываем settype($var, "integer")), возвращает false.
floatval($var)		Функция преобразует переменную $var к вещественному числу.
doubleval($var)		Обе функции совершенно эквивалентны
strval($var)		Функция преобразует переменную $var в строку
intval($var[, $base])	Преобразует переменную $var в целочисленную переменную. По умолчанию числа
			приводятся к привычному нам десятичному формату. Однако при помощи необяза-
			тельного параметра $base можно получить результат, например, в восьмеричном ($base
			= 8) представлении

Кроме указанных функций, в отношении переменных PHP действует C-синтаксис при-
ведения типа, с указанием типа в круглых скобках перед переменной.
$value = 3.14;
echo (int)$value . " (" . gettype((int)$value) . ")"; // 3 (integer)
echo (string)$value . " (" . gettype((string)$value) . ")"; // 3.14 (string)
echo (boolean)$value . " (" . gettype((boolean)$value) . ")"; // 1 (boolean)

(int), (integer)	Приведение к целому типу
(bool), (boolean)	Приведение к логическому типу
(float), (double),	Приведение к вещественному типу
(real)
(string)		Приведение к строке
(array)			Приведение к массиву
(object)		Приведение к объекту
(unset)			Приведение к null
