////////////	SPRING REST		////////////////

Проект Spring REST создается как и обычный Spring Rest через Spring Initialz.

Включаемые зависимости: 
- Spring Web
- Thymeleaf
- Spring Data JPA
- PosthreSQL Driver

При этом, Jackson уже входит в пакет Web.

Так же создается обычный контроллер. 
Помечается также аннотацией @Controller и @RequestMapping("/api").

Но метод контроллера уже помечается аннотацией @ResponseBody, и из нее Spring понимает, что 
в этом методе мы больше не возвращаем название для представления.
В этом методе мы возвращаем данные. 
Клиент получит именно строку, объект класса String который мы вернем.

    @ResponseBody
    @GetMapping("/sayHello")
    public String sayHello() {
        return "Hello World!";
    }
	
Если мы например будем возвращать объект Person, то он будет конвертироваться в json и возвращаться клиенту.
